# NodeJs Testing

@(NodeJs)

`Mocha` 라이브러리를 많이 쓰는 것 같다.

여러가지 기능들이 지원되지만 현재 목적은 코딩 테스트이기 때문에 TDD 하는 방법 정도만 배워보자.

`Mocha`는 자체 assertion 기능을 지원하지 않는다. 
- assertion 하는 라이브러리르 require 하여 사용. 그래서 `chai`를 devDependency로 걸어놓았구나..

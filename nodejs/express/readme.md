#Express

## 무엇인가

- Node.js의 핵심 모듈인 http와 Connect 컴포넌트를 기반으로 하는 웹 프레임워크 (미들웨어)
- express를 사용하지 않는다면 아래와 같은 작업이 반복적으로 이루어짐
    - HTTP 요청 본문 파싱
    - 쿠키 파싱
    - 세션 관리
    - URL 경로와 HTTP 요청 메서드를 기반으로 한 복잡한 if 조건을 통해 라우팅을 구성
    - 데이터 타입을 토대로 한 적절한 응답 헤더 결정
